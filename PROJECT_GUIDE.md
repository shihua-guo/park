# 童行鹏城 - 微信小程序项目说明

这是一个基于微信小程序开发的亲子活动地点推荐应用，提供了深圳地区儿童友好场所的浏览、详情查看和导航功能。

## 📱 页面结构

### 1. 首页 (index)
**路径**: `pages/index/index`

**功能**:
- 使用腾讯地图展示地点位置
- 顶部搜索栏和城市选择
- 分类标签筛选（室内乐园、儿童公园、森林步道等）
- 地图标记点击交互
- 底部地点卡片展示
- 定位功能

**跳转关系**:
- 点击地点卡片 → 跳转到详情页

### 2. 详情页 (detail)
**路径**: `pages/detail/detail`

**功能**:
- 顶部图片轮播展示
- 地点名称、评分和评价数
- 标签展示（空调、年龄段、设施等）
- 地址、营业时间、电话信息
- 地点介绍
- 配套设施展示
- 收藏和评价功能
- 导航按钮

**跳转关系**:
- 点击顶部图片 → 跳转到图片列表页
- 返回按钮 → 返回首页

### 3. 图片列表页 (photos)
**路径**: `pages/photos/photos`

**功能**:
- 瀑布流布局展示多张图片
- 左右两列自适应排列
- 图片懒加载
- 点击图片查看大图

**跳转关系**:
- 点击任意图片 → 跳转到图片详情页
- 返回按钮 → 返回详情页

### 4. 图片详情页 (photo-detail)
**路径**: `pages/photo-detail/photo-detail`

**功能**:
- 全屏展示图片
- 支持左右滑动切换
- 图片计数显示
- 长按保存图片
- 屏幕常亮

**跳转关系**:
- 关闭按钮 → 返回图片列表页

## 🗺️ 腾讯地图集成

### 配置步骤

1. **申请开发者密钥**
   - 访问：https://lbs.qq.com/
   - 注册并创建应用
   - 获取开发者Key

2. **下载SDK**
   - 访问：https://lbs.qq.com/miniProgram/jsSdk/jsSdkGuide/jsSdkOverview
   - 下载最新版本的SDK
   - 替换 `utils/qqmap-wx-jssdk.js` 文件

3. **配置服务器域名**
   - 在微信小程序后台 → 开发 → 开发管理 → 开发设置 → 服务器域名
   - 添加request合法域名：`https://apis.map.qq.com`

4. **使用SDK**
   ```javascript
   const QQMapWX = require('../../utils/qqmap-wx-jssdk.js')
   const qqmapsdk = new QQMapWX({
     key: 'YOUR-KEY-HERE'  // 替换为你的密钥
   })
   ```

## 🎨 设计特色

- **现代化UI**: 采用毛玻璃效果、渐变、阴影等现代设计元素
- **流畅动画**: 页面切换和交互都有平滑的过渡动画
- **响应式布局**: 适配不同尺寸的手机屏幕
- **视觉层次**: 清晰的信息架构和视觉引导

## 🛠️ 技术栈

- 微信小程序原生开发
- 腾讯地图API
- Swiper组件（图片轮播）
- Scroll-view组件（横向滚动）
- 瀑布流布局（CSS Grid）

## 🚀 快速开始

1. **克隆项目**
   ```bash
   cd c:/Users/1/WeChatProjects/miniprogram-1
   ```

2. **配置腾讯地图**
   - 按照上述"腾讯地图集成"步骤配置

3. **准备图标资源**
   - 参考 `images/README.md` 准备地图标记图标

   - 点击编译

   - 测试地图显示
   - 测试页面跳转
   - 测试图片加载

## 🔧 常见问题

### 1. 地图不显示？
- 检查是否配置了腾讯地图Key
- 检查服务器域名是否添加
- 检查定位权限是否开启

### 2. 图片加载慢？
- 使用CDN加速
- 开启图片懒加载
- 压缩图片大小

### 3. 页面跳转报错？
- 检查app.json中pages配置是否正确
- 检查页面路径是否正确
