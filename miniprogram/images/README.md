# 图片资源说明

此目录用于存放小程序所需的图片资源。

## 需要添加的地图标记图标

为了让地图标记正常显示，请准备以下图标文件：

### 1. marker-selected.png
- 尺寸：40px × 40px
- 用途：选中的地点标记（首页地图）
- 建议样式：蓝色圆形，带有图标

### 2. marker-park.png
- 尺寸：35px × 35px
- 用途：公园类地点标记
- 建议样式：绿色圆形，带有公园图标

### 3. marker-forest.png
- 尺寸：35px × 35px
- 用途：森林步道类地点标记
- 建议样式：棕色圆形，带有森林图标

### 4. marker-water.png
- 尺寸：35px × 35px
- 用途：水上乐园类地点标记
- 建议样式：蓝色圆形，带有水滴图标

### 5. marker-farm.png
- 尺寸：35px × 35px
- 用途：农场体验类地点标记
- 建议样式：橙色圆形，带有农场图标

## 图标设计建议

- 使用PNG格式，支持透明背景
- 保持图标简洁清晰
- 使用明亮的配色方案
- 确保在地图上有良好的可见性
- 可以使用Figma、Sketch或在线图标生成工具制作

## 临时方案

在正式图标制作完成前，可以：
1. 使用emoji表情作为临时图标（已在代码中实现）
2. 从图标库网站下载免费图标（如：iconfont、flaticon）
3. 使用微信小程序内置的图标组件
